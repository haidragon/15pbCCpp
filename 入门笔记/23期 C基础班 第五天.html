实现一个函数，需要注意三个点：
1 你定义这个函数的主要功能是什么
2 你实现这个函数，需要什么参数？？？？
3 如果让调用者，得到最终的结果？？？？
比如下面代码：
  1. //比如我们现在要实现一个函数：求得两个数中比较大的数：
  2. int GetMax(int numA, int numB)

  9. 	{
  10. 		return numB;
  11. 	}
  12. }

使用函数的时候应该注意：
1  调用函数的时候，传递的参数类型，要和定义的时候匹配。
2   调用函数的时候，传递的参数个数，要和定义的时候匹配。
什么叫形式参数，什么叫实际参数：
形式参数：定义函数的时候，写的参数
实际参数：调用函数的时候，传递的参数
形式参数改变了，会不会同时改变实际参数的值？？？？
例如：下面的例子，形参的改变并不会影响实际参数的值。
  1. void swap(int numA, int numB)
  2. {
  3. 	int nTemp = 0;
  4. 	nTemp = numA;
  5. 	numA = numB;
  6. 	numB = nTemp;
  7. }
  8. int main(void)
  9. {
  10. 	int a = 10;
  11. 	int b = 20;
  12. 	swap(a, b);
  13. 	printf("a = %d b = %d", a, b);
  14. 	return 0;
  15. }
问：自己定义的函数，是不是只能被main里面的代码调用？？？
答：不是的，除了main函数不能被其他函数调用，一个函数是可以调用另外一个函数的。
例如：下面的例子
  1. void fun2()
  2. {
  3. 	printf("ni hao");
  4. }
  5. void fun()
  6. {
  7. 	fun2();
  8. 	printf("hello world");
  9. }
  11. int main(void)
  12. {
  13. 	fun();
  14. 	return 0;
  15. }
关于局部变量：
  1. //定义在函数内部的变量，我们称之为局部变量。
  2. //局部变量的作用范围（作用域）是在复合语句中。
  3. void fun()

  11. {

  16. }

关于作用域覆盖的问题
  1. int g_NumA = 100;
  2. void fun()
  3. {
  4. 	g_NumA = 200;
  5. 	printf("hello world");
  6. }
  8. int main(void)
  9. {
  10. 	int g_NumA = 0;//当小作用域和大作用域中的变量重名的时候，小作用域的变量起作用
  11. 	g_NumA = 1000;
  12. 	fun();
  14. 	printf("%d", g_NumA);//所以，这里会打印1000
  15. 	return 0;
  16. }
