1.4 extern与static的作用
extern：
声明这个变量或者函数，到其他文件中去寻找。（并不是在定义变量）
注意：
一般一个cpp的供外部使用的函数以及变量，都会声明在相应的.h文件中。
static
作用一：使用在全局变量之前（隐藏）
假如说，本文件中定义的函数或者变量，不想被其他文件使用。就在函数定义前面加上static
作用二：使用在局部变量之前（静态）
  1. void fun()
  2. {
  3. 	//假如一个局部变量前面加上了static，说明这个变量只会被初始化一次。
  4. 	//每次离开函数的时候，还保留着当时的值。
  5. 	//再次进入函数的时候，值是上一次留存下来的。
  6. 	static int Num = 100;
  8. 	printf("%d\n", Num);
  9. 	Num++;
  10. }
  14. int _tmain(int argc, _TCHAR* argv[])
  15. {
  16. 	fun();
  17. 	fun();
  18. 	fun();
  19. 	fun();
  20. 	fun();
  21. 	fun();
  22. 	fun();
  23. 	fun();
  24. 	return 0;
  25. }

2 认识指针
2.1 指针使用的三个步骤
1 定义指针变量：数据类型 * 变量名。 就是定义相应的指针类型的变量
  1. 	int *p1 = NULL;  //定义了一个整型指针
  2. 	char * p2 = NULL;//定义了一个字符型指针
  3. 	short* p3 = NULL;//定义了一个短整型指针
  4. 	//....
2 给指针变量赋值：指针变量全部都是存储地址值的。
  1. 	int *p1 = NULL;  //定义了一个整型指针
  2. 	int nNum = 200;
  3. 	p1 = &nNum;
3 解引用（取内容）
  1. 	int *p1 = NULL;  //定义了一个整型指针
  2. 	int nNum = 200;
  3. 	p1 = &nNum;
  5. 	*p1 = 300;//指针解引用，会访问到指针指向的位置。
  6. 	printf("%d", nNum);//会打印300
2.2 通过指针修改掉实际参数
  1. void fun(int * p)
  2. {
  3. 	*p = 500;
  4. }
  5. int _tmain(int argc, _TCHAR* argv[])
  6. {
  7. 	int nNum = 100;
  8. 	fun(&nNum);
  9. 	printf("%d", nNum);
  10. 	return 0;
  11. }
作用：
1 可以代替返回值。
2 可以在函数内部，修改掉函数外部的变量的值。
