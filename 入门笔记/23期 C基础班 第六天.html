宏-有参宏

  2. int _tmain(int argc, _TCHAR* argv[])
  3. {
  4. 	printf("%lf\n", CALCAREA(6.0));//printf("%lf", 3.14*5.0*5.0);
  6. 	return 0;
  7. }
部分时候，可以代替函数，但是和函数是有区别的，如下面的例子
  1. #define PI 3.14
  2. #define CALCAREA(r) PI*r*r   //这个就叫做有参宏
  4. double CalcArea(double r)
  5. {
  6. 	return PI*r*r;
  7. }
  8. int _tmain(int argc, _TCHAR* argv[])
  9. {
  10. 	printf("%lf\n", CALCAREA(6.0));//printf("%lf", 3.14*5.0*5.0);
  11. 	printf("%lf\n", CALCAREA(5.0+1.0));//printf("%lf", 3.14*5.0+1.0*5.0+1.0);
  12. 	printf("%lf\n", CalcArea(5.0 + 1.0));
  13. 	return 0;
  14. }
 错误的原因：宏是在编译之前，做的纯替换。
一般情况下：
1 代码比较简单，还在程序中大量使用的，我们会定义成有参宏。
2 代码比较复杂，通常都定义成函数。
关于#include的使用：
文件包含预处理#include  功能就是把被包含的文件中的内容赋值粘贴到#include命令的位置。
#include  <>:系统自带的头文件，都使用<>

#include "":自己定义的头文件，都是用""。（系统的，也能使用" "）

但是以上这种定义结构体变量的方式，非常的麻烦。我们可以先构造出类型，然后使用类型名定义变量。
  1. #include <stdio.h>
  2. #include <string.h>
  3. int _tmain(int argc, _TCHAR* argv[])
  4. {
  5. 	//这里是在构造一个结构体类型
  6. 	struct _PEOPLE{
  7. 		int age ;
  8. 		int score;
  9. 		char Name[20];
  10. 	} ;
  11. 	//定义一个结构体变量,初始化结构体变量，放在括号{}中
  12. 	_PEOPLE stcPeople = {18,100,"xiaozhang"};
  13. 	//使用结构体变量
  14. 	stcPeople.age = 18;
  15. 	stcPeople.score = 95;
  16. 	strcpy_s(stcPeople.Name, 20, "xiaoming");
  17. 	return 0;
  18. }

关于联合体：
结构体，每一个成员单独占用内存空间
  1. struct _TEST
  2. {
  3. 	int a ;
  4. 	int b;
  5. };
  6. int _tmain(int argc, _TCHAR* argv[])
  7. {
  8. 	_TEST stc1;
  9. 	stc1.a = 10;
  10. 	stc1.b = 20;
  11. 	printf("%d\n", stc1.a);
  12. 	printf("%d\n", stc1.b);
  13. 	return 0;
  14. }
 联合体，所有的成员共享一块空间

  2. {
  3. 	int a ;

  5. };

  12. 	printf("%d\n", stc1.b);
