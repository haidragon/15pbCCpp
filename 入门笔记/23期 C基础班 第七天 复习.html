3 堆空间
申请一块内存：申请出内存的首地址  malloc（要申请多大，以字节为单位）；
释放一块内存：free
注意：释放的必须是被申请出的内存。
  1. int _tmain(int argc, _TCHAR* argv[])
  2. {
  3. 	int Array[20] = {1,2,3,4,5,6};
  4. 	int nNum = 100;
  5. //	free(&nNum);//不行的，不能释放栈空间（自己定义的普通局部变量，局部数组）
  7. 	int *P = (int *)malloc(10 * sizeof(int));
  9. 	free(P);
  10. 	//....
  11. 	free(P);//一个申请出的地址不能二次释放。
  12. 	return 0;
  13. }
其他操作内存的函数：
memset（起始地址，要设置的值，要设置多大）：一般用于给刚申请出的内存，设置一个初识值。
memcpy（目标地址，源数据地址，要拷贝多大）：一般用于要把内存中的数据考到另一块内存中
