4. 函数重载

要求：同名函数，参数的类型、个数、顺序3者满足1个不同即构成函数重载

5. 默认参数

赋值规则：从右向左，中间不能隔断 
如果左边的参数有默认值，右边的肯定有

约定（不是强制要求）：函数声明和函数定义，默认参数只能写在一个里，要么写在声明里，要么写在定义里。一般约定写在声明里，因为声明是给别人看的，得让用的人知道默认参数是什么

6. 引用

不占内存，必须初始化，初始化后的赋值是什么意思： 
```cpp 
int a = 10; 
int &b = a;// 初始化b，让b成为a的小名，后面所有对b的操作都是对a的操作，对a或b的修改都会影响b或a 
int c = 200; 
b = c;// 把c的值给a和b 
&b = c;// error 
printf("%p"，&b);// b的地址就是a的地址

```
