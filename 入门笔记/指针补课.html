## 指针定义
指针：变量，干什么用的：保存地址的
定义：保存的地址的类型 * 标识符
int * p;// p用来保存int型数据的地址
int nNum = 10;
p = &nNum;
char c;
p = &c;// error p只能用来保存int型数据地址，c是char型数据

## 指针的运算
指针只能做加减常量的运算，指针之间不做算数运算，因为没意义
指针做加减运算的时候，加减的是单位，单位大小取决于指针类型
```cpp
int nNum = 10;
// *和类型在一起的时候表示指针类型
int *p = &nNum;// 表示p = &nNum;
// 当不和类型在一起的时候表示解引用
*p = 100;// 表示把nNum的值赋值为100 nNum = 100
// 指针的加减运算
p+1;// 加的是1单位  大小取决于指针类型  int型加4
```
## 指针和数组的关系
一级指针和1维数组等价
```cpp
int a[5] = {1,2,3,4,5};
// 数组名a表示1维数组的地址
int *p = &a[0];// a和&a[0]等价 所以可以写成 int *p = a;
// 对指针的解引用操作
*p == a[0];// true
p+1 == &a[0] + 1 == &a[1];
*(p+1) == a[1];
*(p+x) == a[x];
*(p+x) == p[x];  p[x] == a[x]; p == a;
```
## 二维数组
```cpp
// 数组：同类型元素的集合
// 1维数组
int a[10];// int类型的数据的集合
// 2维数组
// int[3]数据类型的集合
// int[3]是1维数组
// b就是同类型的（int[3]）的集合，集合元素有2个
// b有两个元素b[0] b[1]，这两个元素都是int[3] 是1维数组
int b[2][3];
b[0]+1// a+1  加4字节 == 1元素单位 它的元素是整型int
b+1;// b是2维数组 +1 元素 b是1维数组的集合 它的元素就是1维数组 1单位就是1维数组的大小 4*3
```


## 文件操作
```cpp
FILE* pFile = nullptr;
fopen_s(
    &pFile,// 文件指针
    "123.txt",// 要打开的文件路径
    "r+");// 以读写方式打开文件，文件指针在文件开始处
fread(p,// 要读到哪去
    size,// 缓冲区p的大小
     一次读多少,
     读几次,
     pFile);// 从哪个文件读
// 移动文件指针到你要写入的位置
fseek(pfile,// 移动哪个文件的指针
     0,// 距离第3个参数的偏移值
     seek_set);// 从文件的哪个位置开始计算偏移
fwrite(从哪个缓冲区读数据写到文件里,
      一次写多少,
      写几次,
      从哪个文件写);
fclose(pfile);
```
1. `fopen`打开文件
2. `fclose`关闭文件
3. `fread fscanf`读取文件
4. `fwrite fprintf`写入文件
5. `fseek`移动文件指针

- 示例
```cpp
int main()
{
    FILE *pFile = nullptr;
    // 打开文件
    fopen_s(&pFile, "d:\\123.txt", "r+");
    // 把文件指针移动到最后
    fseek(pFile, 0, SEEK_END);
    // 告诉我文件位置获取文件大小
    int nSize = ftell(pFile);
    // 写入文件
    char*p = "123";
    fwrite(p, 3, 1, pFile);
    // 移动文件指针到写之前的位置
    fseek(pFile, -3, SEEK_CUR);
    char sz[10] = {};
    fread_s(sz, 10, 3, 1, pFile);
    printf("%s", sz);
    return 0;
}
```
